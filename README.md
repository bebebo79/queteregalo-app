# 🎁 QueTeRegalo

Aplicación web full-stack para gestionar ideas de regalos de forma sencilla.

El objetivo del proyecto es permitir a los usuarios guardar, organizar y consultar ideas de regalos para diferentes personas y ocasiones.

Este proyecto forma parte de mi aprendizaje en desarrollo **full-stack con JavaScript/TypeScript**.

---

## 🚀 Tecnologías utilizadas

### Backend
- Node.js
- Express
- TypeScript
- PostgreSQL

### Frontend
- React
- JavaScript
- CSS

## ⚙️ Instalación del proyecto

Clonar el repositorio:

```bash
git clone https://github.com/bebebo79/queteregalo-app.git
cd queteregalo-app
```

---

## ▶️ Ejecutar el backend

```bash
cd QueTeRegalo_backend
npm install
npm run dev
```

El backend se ejecutará normalmente en:

```
http://localhost:3000
```

---

## ▶️ Ejecutar el frontend

En otra terminal:

```bash
cd QueTeRegalo_frontend
npm install
npm start
```

El frontend se abrirá en:

```
http://localhost:4200
```

---

## 🗄️ Variables de entorno

El backend requiere un archivo `.env` con la conexión a la base de datos.

Ejemplo:

```
MONGO_URI=your_database_connection
PORT=3000
```
